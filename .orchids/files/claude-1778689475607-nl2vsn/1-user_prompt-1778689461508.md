# BRAND OVERVIEW



## Brand Name
KASSY BAKES



## Brand Type
Homemade cat-themed boutique bakery brand.



## Core Brand Identity
Kassy Bakes is not just a bakery website.  
It is a cozy handmade dessert experience built around warmth, emotional storytelling, homemade aesthetics, and two mascot cats named Rico and Sam.



The brand should feel:



- warm
- cozy
- handmade
- emotionally personal
- playful but elegant
- café-inspired
- premium homemade
- visually memorable



The brand should NEVER feel:



- corporate
- sterile
- minimalist tech startup
- luxury fashion brand
- hyper-cartoonish
- childish
- over-saturated pastel
- generic ecommerce bakery



The emotional goal is:



"A cozy homemade bakery run by a passionate cat lover."
